Overview
This Weather App is a simple React application that allows users to input a city name and retrieve the current weather data for that city. The application fetches data from a weather API and displays it in a user-friendly interface.

Features
Search weather by city name
Display current temperature, humidity, wind speed, and weather conditions
Error handling for invalid city names
Technologies Used
    React
    JavaScript (ES6+)
    HTML5
    CSS3
    OpenWeatherMap API (or any other weather API of your choice)
